# About Schemasense

Link to Live Project: https://schemasense.vercel.app/

Schemasense is a full-stack database intelligence platform built on FastAPI and React that connects to enterprise databases to automatically generate intelligent documentation and insights. It transforms complex database structures into comprehensive, readable documentation while providing real-time quality metrics and data profiling. The platform includes an interactive chat interface that enables users to ask business questions and retrieve instant database insights without writing SQL queries.
